# AI-DiaCar
Данный дипломный проект является веб-приложением с машинным обучением для диагностики автомобилей с подробными данными об автомобилях, их неисправностях и особенностях систем, который опирается на оригинальные мануалы конкретных моделей автомобилей.
# Backend
Backend - API для быстрой и безопасной обработки данных пользователей, автомобилей и другой необходимой информации для работы веб-приложения. 
- Java, Spring Boot, Jakarta.
# Frontend
Frontend - внешняя оболочка работающего веб-приложения.
- JavaScript, NodeJS,  React.
# AI Model
AI Model - модель машинного обучения с учителем, обучение происходит на основе огромного количества данных, полученных из оригинальных мануалов и достоверных источников в формате PDF.
- Python
